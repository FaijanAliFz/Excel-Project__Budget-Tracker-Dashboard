## 📊 Project Overview
An interactive **Budget Tracker Dashboard** built to monitor income, expenses, and savings in a structured and visual format. It converts raw financial data into actionable insights for better financial planning.

---

## 🎯 Objectives
- Track total income and expenses  
- Analyze category-wise spending patterns  
- Monitor monthly financial trends  
- Identify overspending areas  
- Improve budgeting and savings decisions  

---

## ⚙️ Dashboard Features
- Dynamic and interactive charts  
- Category-wise expense analysis  
- Monthly income vs expense comparison  
- Automated savings calculation  
- Clean and user-friendly layout  

---

## 🛠️ Tools & Techniques
- Microsoft Excel  
- Pivot Tables & Pivot Charts  
- Data Cleaning & Formatting  
- Conditional Formatting  
- Data Validation (Dropdown filters)  
- Excel Formulas (SUM, IF, Percentage calculations)  

---

## 📈 Key Insights
- Identified highest spending categories  
- Tracked monthly income and expense trends  
- Highlighted savings patterns over time  
- Enabled quick decision-making using visual KPIs  

---

## 🚀 Project Outcome
Converted raw financial data into a **clear and interactive dashboard** for better expense tracking and smarter financial decisions.
